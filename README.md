### Getting Started with v-gsap-nuxt

#### Install manually
1. Install from npm

``` 
npm i v-gsap-nuxt
```
2. Add module to your ```nuxt.config.ts```

```
modules: [
  'v-gsap-nuxt'
]
```

and follow the [doc](https://v-gsap-nuxt.vercel.app/installation/philosophy).
